Sortowanie przez wybieranie

Ustawiamy uczniów według wzrostu, krok po kroku.

Krok: 0 Gotowe do startu.
Wskazanie: najedź myszą na ucznia.

Co tu obserwujesz?

Krok 1: ustawiamy wskaźnik i na pierwszej nieuporządkowanej pozycji.

Krok 2: szukamy najmniejszego elementu w pozostałej części (wskaźnik j).

Krok 3: zamieniamy znaleziony najmniejszy z elementem na pozycji i.

Na ekranie: żółty to i, zielony to j, czerwony to aktualne minimum.

Przykład z życia: wybierasz najniższego ucznia do ustawienia na początku, potem znów szukasz najniższego z pozostałych.

aktualna pozycja i sprawdzany uczeń j najmniejszy znaleziony część posortowana
Dźwięki: niski = porównanie, wyższy = zamiana.

10 for (int i = 0; i < N - 1; i++) {
11 int min = i;
12 for (int j = i + 1; j < N; j++) {
13 if (a[j] < a[min]) min = j;
14 }
15 if (min != i) swap(a[i], a[min]);
16 }
17}
