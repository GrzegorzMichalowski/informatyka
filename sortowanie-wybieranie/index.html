Sortowanie przez wybieranie

Ustawiamy uczniów według wzrostu, krok po kroku.

Krok: 0 Gotowe do startu.
Wskazanie: najedź myszą na ucznia.
aktualna pozycja i sprawdzany uczeń j najmniejszy znaleziony część posortowana

C++ (z komentarzami)

1// Sortowanie przez wybieranie
2void selectionSort(vector<int>& a) {
3 int n = a.size();
4 for (int i = 0; i < n - 1; i++) { // wybieramy pozycję i
5 int minIdx = i; // zakładamy, że i jest najmniejsze
6 for (int j = i + 1; j < n; j++) { // szukamy mniejszego
7 if (a[j] < a[minIdx]) { // porównanie
8 minIdx = j; // zapamiętujemy nowy minimum
9 }
10 }
11 if (minIdx != i) swap(a[i], a[minIdx]); // zamiana
12 }
13}
