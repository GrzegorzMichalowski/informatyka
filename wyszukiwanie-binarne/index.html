Wyszukiwanie binarne

Szukamy elementu w uporządkowanym zbiorze przez połowienie.

Dane do wyszukania

Tablica uporządkowana (do wyszukania)
Krok: 0 Gotowe do startu.

Co tu obserwujesz?

Krok 1: wybieramy element środkowy (s).

Krok 2: porównujemy go z szukaną wartością.

Krok 3: odrzucamy połowę listy i powtarzamy.

Na ekranie: granice l, p i środek s.

Przykład z życia: szukasz nazwiska na alfabetycznej liście — otwierasz w połowie i odrzucasz połowę stron.

Dźwięki: niski = porównanie, wyższy = znaleziono.

C++ (styl szkolny)

1const int N = 5;
2int a[N];
3int SzukajBinarnie(int x) {
4 int l = 0, p = N - 1;
5 while (l <= p) {
6 int s = (l + p) / 2;
7 if (a[s] == x) return s;
8 if (a[s] < x) l = s + 1; else p = s - 1;
9 }
10 return -1;
11}
