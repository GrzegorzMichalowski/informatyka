Sortowanie bąbelkowe

„Przepychamy” największe elementy na koniec, krok po kroku.

Krok: 0 Gotowe do startu.
Wskazanie: najedź myszą na ucznia.
aktualna para porównywana element po lewej element po prawej część posortowana
Dźwięki: niski = porównanie, wyższy = zamiana.

C++ (z komentarzami)

1// Sortowanie bąbelkowe
2void bubbleSort(vector<int>& a) {
3 int n = a.size();
4 for (int i = 0; i < n - 1; i++) { // kolejne przebiegi
5 for (int j = 0; j < n - i - 1; j++) { // porównujemy pary
6 if (a[j] > a[j + 1]) { // jeśli kolejność zła
7 swap(a[j], a[j + 1]); // zamiana sąsiadów
8 }
9 }
10 }
11}
