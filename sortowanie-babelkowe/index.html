Sortowanie bąbelkowe

„Przepychamy” największe elementy na koniec, krok po kroku.

Dane do uporządkowania

Krok: 0 Gotowe do startu.
Wskazanie: najedź myszą na ucznia.

Co tu obserwujesz?

Krok 1: porównujemy sąsiednie elementy od lewej do prawej.

Krok 2: jeśli są w złej kolejności, zamieniamy miejscami.

Krok 3: po jednym przebiegu największy element „wypływa” na koniec.

Na ekranie: zielony to lewy element pary, niebieski to prawy.

Przykład z życia: przepychasz największe książki na koniec półki, przesuwając je po kolei.

aktualna para porównywana element po lewej element po prawej część posortowana
Dźwięki: niski = porównanie, wyższy = zamiana.

C++ (z komentarzami)

1const int N = 5;
2int a[N];
3void WprowadzDane() {
4 for (int i = 0; i < N; i++) {
5 cout << \"Podaj dane nr \" << i << \": \";
6 cin >> a[i];
7 }
8}
9void SortujBabelkowo() {
10 for (int i = 0; i < N - 1; i++) {
11 for (int j = 0; j < N - i - 1; j++) {
12 if (a[j] > a[j + 1]) swap(a[j], a[j + 1]);
13 }
14 }
15}
